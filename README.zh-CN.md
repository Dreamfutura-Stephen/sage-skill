# 智者 Skill（Wise Sage）

> 智者 Skill 把历史上的智者——庄子、苏格拉底、孔子、尼采、王阳明……——蒸馏为**可对话的思维训练伙伴**。
>
> 不是聊天机器人。不是语录搜索引擎。而是**苏格拉底式的思维陪练**：反问你的假设、暴露你的盲区、给你新的认知框架——然后让你自己找到答案。

## 四种对话模式

| 模式 | 用法 | 说明 |
|------|------|------|
| 🧪 **蒸馏** | `蒸馏 庄子` | 生成完整的智者认知档案：第一性原理、核心思维框架、内在张力、边界声明、智识谱系、已知批评 |
| 💬 **对话** | `与 孔子 对话` | 苏格拉底式问答：反弹→暴露假设→框架注入→边界提醒，四层递进 |
| ⚖️ **比较** | `比较 尼采 vs 加缪 关于自由` | 两种思维框架在同一问题下的同与异 |
| 🪑 **圆桌** | `圆桌 人生的意义 邀请 庄子 加缪 王阳明` | 跨时空思想碰撞，多位智者从各自框架回应同一问题 |

> 💡 **不知道找谁聊？** 说出你的困惑，Skill 会推荐 2–3 位最适合的智者。

## 核心理念

1. **不给答案，给镜子和梯子**——提供认知工具，不是结论
2. **合适时引用原典，不合适时按思维框架生成**——不强行塞入不相关原典
3. **永远暴露边界**——每次回答都附带时代局限
4. **五条硬限制：** 不碰政治、不编造、不包装通用道理、不隐藏争议、信息不足不强行生成
