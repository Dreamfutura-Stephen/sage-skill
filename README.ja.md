# 智者 Skill（ワイズセージ）

> 歴史上の賢人たち——荘子、ソクラテス、孔子、ニーチェ、王陽明、アインシュタイン、チャーリー・マンガー……——を**対話可能な思考トレーニングパートナー**として蒸留します。
>
> チャットボットではありません。名言検索エンジンでもありません。**ソクラテス式の思考トレーナー**です：あなたの前提を問い返し、盲点を暴き、新しい認知フレームワークを提供し——そしてあなた自身で答えを見つけさせます。

## 4つのモード

| モード | 使い方 | 効果 |
|--------|--------|------|
| 🧪 **蒸留** | `蒸留 荘子` | 賢者の完全な認知プロファイルを生成：第一原理、思考枠組み、内的緊張、境界声明、知的系譜 |
| 💬 **対話** | `孔子 と対話` | ソクラテス式問答：**跳ね返し**→**前提暴露**→**枠組み注入**→**境界提醒** |
| ⚖️ **比較** | `比較 ニーチェ vs カミュ 自由について` | 二つの思考枠組みを同一問題で比較 |
| 🪑 **円卓** | `円卓 人生の意味 招待 荘子 カミュ 王陽明` | 時空を超えた思想の衝突 |

> 💡 **誰に話しかければいいかわからない？** あなたの悩みを伝えてください。スキルが最適な賢者を2〜3人推薦します。

## 核心理念

1. **答えではなく、鏡と梯子を**——認知的ツールを提供し、結論は押し付けない
2. **原典を適切に引用し、そうでない場合は思考枠組みで生成**——無関係な原典を無理に引用しない
3. **常に境界を明示**——すべての回答に時代的限界を付記
4. **五つの制限：** 政治不議論、捏造禁止、一般論を独自見解と偽らない、論争を隠さない、情報不足では生成しない
