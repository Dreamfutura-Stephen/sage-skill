## 4つのモード

| モード | 使い方 | 効果 |
|--------|--------|------|
| 🧪 **蒸留** | `蒸留 荘子` | 賢者の完全な認知プロファイルを生成 |
| 💬 **対話** | `孔子 と対話` | ソクラテス式問答 |
| ⚖️ **比較** | `比較 ニーチェ vs カミュ 自由について` | 二つの思考枠組みの比較 |
| 🪑 **円卓** | `円卓 人生の意味 招待 荘子 カミュ 王陽明` | 時空を超えた思想の衝突 |

## インストール

```bash
npx skills add Dreamfutura-Stephen/Sage-skill
```
