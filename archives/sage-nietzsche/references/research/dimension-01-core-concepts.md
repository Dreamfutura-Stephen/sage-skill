# 维度一：原著精要 —— 尼采核心概念

## 1. 超人 (Ubermensch / Overman)

- 出自《查拉图斯特拉如是说》序言
- 尼采对理想人类的构想：人应被克服，"人是应被超越的某种东西"
- 不是"超人"字面意义的超级英雄，而是**自我创造价值、肯定生命轮回**的人
- 与"末人"(Last Man)形成鲜明对立——末人追求舒适、安全、小确幸，"发明了幸福"却失去了创造意志
- 三重变形：骆驼（承载）→ 狮子（否定"你应"）→ 孩子（神圣的"是"）

## 2. 权力意志 (Will to Power / Wille zur Macht)

- 尼采回答"生命本质是什么"
- 不是政治权力，而是**一切生命追求自我克服、成长、扩张的驱动力**
- 核心公式："生命本身就是那种必须不断克服自身的东西"
- 本体论层面：实在不是静态的"存在"(Being)，而是永恒的"生成"(Becoming)
- 价值创造功能：强者创造价值，弱者通过怨恨(ressentiment)反应

## 3. 永恒轮回 (Eternal Return)

- 《查拉图斯特拉如是说》第三部"幻象与谜"中的核心思想
- 不仅仅是宇宙论假说，更是**生存论测试**："假如有个恶魔对你说——你愿意重复这生活无数次吗？"
- 门道（Moment）意象：过去与未来两条路在此交汇，时间是个圆
- 牧羊人咬断喉咙里的蛇头：克服对轮回的恶心与恐惧
- Amor Fati（爱命运）：对生命的完全肯定

## 4. 上帝死了 (God is Dead)

- 出自《快乐的科学》狂人段落
- 并非简单的无神论宣言——而是**西方形而上学-道德基础的崩溃**
- 狂人的故事：市场里的人嘲笑他——他们"不信上帝"却不知后果的严重性
- 后果：虚无主义危机——"最高价值自行贬值"
- 科学杀死了上帝，但科学本身也失去了形而上学的根基
- 当代意义：后真相时代的预言、制度权威的危机

## 5. 主人道德 vs 奴隶道德

- 出自《道德的谱系》
- **主人道德**：起源于贵族/强者，"好"="高贵/强大"，"坏"="低贱/弱小"
- **奴隶道德**：起源于被压迫者，通过**怨恨(ressentiment)**反转价值
- 价值重估："好"变为"温和/谦卑/同情"，"恶"变为"骄傲/强大/自信"
- 犹太-基督教作为"奴隶道德的反叛"
- 注意：尼采不是简单支持主人道德——他要的是**超越奴隶道德后的全新创造**

## 6. 酒神 vs 日神

- 出自《悲剧的诞生》
- **日神(Apollonian)**：个体化、形式、梦境、边界、理性的美
- **酒神(Dionysian)**：消解个体、醉态、狂欢、超越界限、原始的生命力量
- 希腊悲剧的伟大在于**二者的结合**
- 后期尼采：酒神哲学家——酒神成为生命肯定的最高象征
- 苏格拉底/理性主义杀死了悲剧精神

## 核心概念的关系网络

权力意志（驱动力）-> 永恒轮回（测试）-> 超人（肯定者）-> 价值重估（行动）

- 上帝死了 -> 虚无主义危机
- 超越虚无主义 -> 超人通过权力意志创造新价值
- 肯定永恒轮回 -> 最彻底的"是"（amor fati）
- 主人/奴隶道德 -> 谱系学揭示道德的起源
- 酒神/日神 -> 文明与创造力的两极张力
