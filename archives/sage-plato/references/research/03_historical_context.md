# 维度三：历史语境

## 生平时间线

| 日期 | 事件 |
|------|------|
| c. 428-427 BCE | 柏拉图出生于雅典贵族家庭 |
| 431-404 BCE | 伯罗奔尼撒战争 |
| 404-403 BCE | 三十僭主统治（柏拉图亲属参与）|
| 403 BCE | 民主体制恢复 |
| c. 407 BCE | 拜苏格拉底为师，烧毁早年诗歌作品 |
| 399 BCE | **苏格拉底被处死** |
| 399-388 BCE | 漫游（麦加拉、埃及、意大利）|
| c. 388 BCE | **第一次叙拉古之旅**——试图说服统治者建立哲学王国家；返回途中被卖为奴隶 |
| c. 387 BCE | **创立学园（Academy）**——西方最早的高等教育机构 |
| 367 BCE | **第二次叙拉古之旅**——应狄奥尼修斯二世邀请；最终扫兴而归 |
| 361-360 BCE | **第三次叙拉古之旅**——再次失败 |
| c. 348-347 BCE | 柏拉图去世，享年约80岁 |

## 关键历史背景

### 雅典民主的衰败对柏拉图的影响
- 伯罗奔尼撒战争雅典战败，民主体制动摇
- 三十僭主的暴政（柏拉图亲属恰米德斯和克里底亚参与其中）
- 民主恢复后处死苏格拉底——这使柏拉图彻底对民主失望

### 苏格拉底之死的决定性影响
- 判决：不敬神、败坏青年——由约500人陪审团以280:220票定罪
- 苏格拉底拒绝认罪或流亡，饮鸩而死
- 柏拉图不在现场（《斐多篇》记载），但对柏拉图影响深远
- 使柏拉图得出结论：**除非哲学家成为统治者，否则人类罪恶不会停止**
- 成为柏拉图一生奋斗的"哲学抗争"原型

### 三次叙拉古之旅的教训
- 第一次：直接政治改造的尝试失败，险遭不测
- 第二次：教育僭主狄奥尼修斯二世的努力失败
- 第三次：彻底的政治理想幻灭
- 标志着柏拉图从"直接参与政治"转向"通过教育与著述影响后世"

### 学园（Academy）的创立
- 位于雅典城郊阿卡德米运动场附近
- 门口挂牌："不懂几何学者免进"
- 课程：数学、天文、音乐、哲学
- 兼具教学与研究功能，被认为是欧洲最早的大学前身
- 被称为"政治训练班"
- **延续900多年**，直至529年被东罗马皇帝查士丁尼关闭
- 最著名学生：亚里士多德（后创立吕克昂学园）

### 与智者学派和犬儒学派的竞争
- 智者学派：以普罗泰戈拉、高尔吉亚为代表，收费教学修辞术。柏拉图在对话中反复驳斥智者的相对主义和诡辩，认为他们以意见冒充真知
- 犬儒学派：第欧根尼为代表的极端简朴生活。柏拉图与犬儒学派在价值观上有根本分歧——柏拉图追求理型世界，犬儒学派拒斥一切社会习俗和形而上学
