# 维度三：历史语境——孔子回应什么

## 一、春秋末期的时代背景

### 1.1 "礼崩乐坏"的具体表现

孔子（前551—前479）生活于春秋末期，周王室衰微到几乎失去实际控制力，诸侯争霸，卿大夫专权乃至"陪臣执国命"。

**制度僭越（六种具体表现）**：

| 事件 | 记载 | 孔子反应 |
|------|------|---------|
| 季氏八佾舞于庭 | 大夫用天子规格64人舞队 | "是可忍也，孰不可忍也" |
| 三家者以雍彻 | 大夫撤祭唱天子祭诗 | "相维辟公，天子穆穆，奚取于三家之堂" |
| 季氏旅于泰山 | 大夫祭名山大川 | "曾谓泰山不如林放乎" |
| 鲁禘祭中逆祀 | 颠倒闵公僖公位次 | "自既灌而往者，吾不欲观之矣" |
| 礼乐形式化 | "礼云礼云，玉帛云乎哉" | 批判内在精神的丧失 |
| 礼乐征伐自诸侯出 | 权力全面下移 | "天下无道" |

### 1.2 鲁国的特殊政治格局

- 鲁国为周公封地，保存了最完整的周礼典章
- 但政权实际由"三桓"（季孙、叔孙、孟孙三家，鲁桓公后裔）控制
- 到孔子时期，三桓的家臣（阳虎、公山不狃）又开始控制大夫采邑
- 孔子从政的"堕三都"正是针对这种"权力下移链条"

## 二、孔子从政经历

### 2.1 仕鲁时期（约前501—前497，孔子51-55岁）

| 职务 | 时间 | 主要政绩 |
|------|------|---------|
| 中都宰 | 前501 | 治理一年，"路不拾遗、夜不闭户" |
| 小司空→大司寇→摄行相事 | 前500 | 夹谷之会（前500）：以礼斥齐国劫持，收复郓、讙、龟阴三地 |
| 堕三都 | 前498 | 叔孙氏郈邑、季孙氏费邑成功堕毁；孟孙氏成邑抵抗未堕，半途而废 |

### 2.2 堕三都失败的分析

- **表面原因**：孟孙氏家臣公敛处父的武装抵抗
- **深层原因**：三桓意识到"堕都"实际上是"强公室、弱私家"——削弱三家以加强鲁君权力
- **后果**：三桓与孔子矛盾公开化，孔子失去政治基础
- **导火索**：齐国送女乐，鲁定公和季桓子沉溺声色，郊祭后不送祭肉——政治信号明确，孔子出走

### 2.3 周游列国（前497—前484，14年）

| 目的地 | 经历 | 关键事件 |
|--------|------|---------|
| 卫国（多次往返） | 停留最久，俸禄六万粟但不任实职 | 匡地被困（被误认为阳虎）；见南子引发争议；卫灵公问阵孔子答"军旅未学" |
| 曹国 | 短暂停留未被重用 | — |
| 宋国 | 司马桓魋追杀 | "天生德于予，桓魋其如予何"；微服过宋 |
| 郑国 | 与弟子走散 | 被形容"累累若丧家之狗"，孔子欣然大笑 |
| 陈国 | 居住三年 | 陈国战乱 |
| 蔡国→楚国（途中） | 陈蔡之厄 | **绝粮七日**，"从者病，莫能兴"，孔子弦歌不辍 |
| 楚国 | 楚昭王欲封地，被子西劝阻 | 未能在楚施展抱负 |
| 返回卫国→鲁国 | 弟子冉有助其归鲁 | 尊为"国老"但不任实职 |

## 三、同时代的思想对话者

### 3.1 孔子与老子

- **问礼于老子**：《史记》《孔子家语》等记载孔子适周问礼
- 老子告诫："良贾深藏若虚，君子盛德容貌若愚"
- 孔子评价老子"犹龙"
- **思想差异**：老子"无为" vs 孔子"有为"；老子"绝仁弃义" vs 孔子"仁义为本"

### 3.2 少正卯——诛少正卯的学术争议

| 观点 | 主要论据 | 代表人物 |
|------|---------|---------|
| **实有说** | 《荀子·宥坐》《史记》《孔子家语》等多有记载 | 荀子、司马迁、胡适 |
| **伪造说** | 《论语》《左传》《孟子》均不载；首见于200年后的《荀子》；"刑不上大夫"体制矛盾 | 徐复观、部分现代学者 |

**思想史意义**：争议本身反映了后世对孔子形象的投射——"子为政焉用杀"的孔子形象与诛杀行为的矛盾，迫使历代学者不断重新解释这一事件。

### 3.3 同时代的隐逸批评者

- **楚狂接舆**：歌而过孔子曰"凤兮凤兮，何德之衰"（《微子》5）
- **长沮、桀溺**："滔滔者天下皆是也，而谁以易之"（《微子》6）
- **荷蓧丈人**："四体不勤，五谷不分，孰为夫子"（《微子》7）
- **晨门（守门人）**："是知其不可而为之者与"（《宪问》38）

这些隐逸者对孔子的评价（知其不可而为之、不合时宜）恰恰点出了孔子最本质的精神特征。

## 四、孔子颠覆了什么"默认答案"

1. **宗教天命→道德天命**：以前的天命是神灵意志，孔子将其转化为道德使命，"天"开始具有道德含义
2. **固定等级→可修养的等级**："君子"从血统身份变成道德修养目标——"人皆可以为尧舜"
3. **知识垄断→有教无类**：将原本贵族垄断的"六艺"教育向平民开放
4. **形式化的礼→注入仁的礼**：礼不再是机械执行的仪式，而是内在情感的外化
5. **武力征服→以德治国**：在霸道盛行的时代坚持王道理想

---

## 五、孔子思想的"进攻性"——挑战了谁的权力

1. **挑战世袭贵族**："学而优则仕"冲击了"血缘即等级"的铁律
2. **挑战君主专断**：以"道"高于"君"——"以道事君，不可则止"（《先进》24）
3. **挑战法家的严刑峻法路径**：在齐桓公、管仲的"霸道"成功实践面前坚持"王道"
4. **挑战隐逸者的避世立场**：坚持"鸟兽不可与同群，吾非斯人之徒与而谁与"

## 备注

- "礼崩乐坏"不能简单理解为"道德堕落"——它是新旧权力结构转换过程中的必然现象
- 孔子不是要简单地"回到过去"，他的"复礼"实质上是对礼进行了"仁"的重新奠基
