# 维特根斯坦认知档案 · 六维研究
## 维度二：思辨方法

---

### 一、"不要想，要看！"（Don't Think, But Look!）

#### 语境
这一名言出现在《哲学研究》§66，紧随家族相似的讨论。维特根斯坦在这里批评了传统哲学的方法论假设——认为概念必然具有某种共同的"本质"或"特征"。

#### 方法论内涵
1. **从理论转向描述**：哲学不应该构建理论，而应该描述语言的实际使用。维特根斯坦说："我们不会提出任何理论……我们必须抛弃一切解释，而仅用描述来取而代之。"（PI §109）
2. **"看"不是感性直观，而是观察用法**：不是诉诸内省或神秘洞见，而是仔细审视语言在实际生活中的运作。
3. **拒斥"深层/表面"二分法**：哲学问题不是藏在表面之下的深层之谜，而是在表面上的混淆——"哲学的问题是当我们让语言放假时产生的"（PI §38）。
4. **"没有什么是隐藏的"（Nothing is hidden）**：哲学困惑的解答就在日常语言的使用之中，不需要特殊的知识或理论。

### 二、治疗式哲学（Philosophy as Therapy）

#### 核心论题
维特根斯坦多次将哲学方法比作治疗：
- **PI §255**："哲学家的处理问题方式，如同治疗疾病。"
- **PI §133**："没有单一的哲学方法，但有多种方法，如同不同的治疗方式。"
- **PI §593**："哲学疾病的一个主要原因——单一化的饮食：只用一种例子来滋养思想。"

#### 哲学的"疾病"是什么？
- **哲学问题不是真正的理论问题，而是困惑（Befremdung）**。
- 产生原因：语言被从日常使用中剥离出来，"当语言放假时"，产生了形而上学的幻觉。
- 哲学家被"图像"捕获——比如"心灵是一个内在领域"这幅图像——然后试图解决因此产生的问题。

#### 治疗的"疗法"是什么？
1. **诊断**：揭示哲学问题源于语言的误用和范畴混淆。
2. **澄清**：将词语从形而上学使用"带回"日常使用。
3. **消解而非解决**：问题不是被回答了，而是被看出是伪问题，从而"消失"。
4. **让哲学达到安宁**（PI §133）：哲学的真正目标是"让哲学飞轮停止转动"——不是产生新知识，而是消除混乱。

#### 关于"治疗"的学术争论
- **隐喻说**（主流观点）：大多数评论者认为"治疗"是比喻，指通过概念澄清来消解哲学问题。
- **字面说**：Eugen Fischer（《哲学幻觉及其治疗》）主张哲学问题**字面上**是一种幻觉，可以通过认知科学、语言学和心理学来诊断。
- **怀疑说**：部分学者（如DOAJ文章"Why Later Wittgenstein was not a therapist"）认为维特根斯坦的目标仍然是**解决**哲学问题，治疗式解读被夸大了。
- **Coliva的中间立场**：在《论确定性》中，维特根斯坦旨在表明怀疑论疑虑是误入歧途，而非仅仅提供治疗。

### 三、把哲学问题拆解为语言混淆

#### 操作方法
维特根斯坦的哲学方法可以总结为一系列操作：
1. **寻找语法幻觉**：揭示哲学问题如何源于对语言语法的误解。例如，将名词（"心灵"、"时间"）理解为指称实体，而实际上它们只是语法主语。
2. **构造过渡性命题（Übergangssätze）**：提出"荒谬的"中间命题，暴露所使用的概念框架的不连贯性。
3. **引出自然反应**：将哲学家拉回到日常语言的"地球"上——"让我们看看在日常生活中我们是怎样使用这个词的。"
4. **使用类比和比喻**：大量的比喻（甲虫、梯子、纺车、苍蝇瓶、河床）不是装饰，而是方法论工具。

#### 经典案例分析
- **"椅子在我离开房间时真的存在吗？"**——这不是一个真正的形而上学问题，而是哲学家混淆了"看"和"想象"两种语言游戏。
- **"疼痛是私人感觉"**——这不是一个理论发现，而是对"疼痛"一词的语法的混淆。当我们说"我有疼痛"时，我们不是在报告一个私人对象。

### 四、示范而非论证（Showing, Not Arguing）

#### 方法论核心
维特根斯坦不采用传统哲学中标准的论证形式（前提→结论的推理）。他的文本由以下组成：
- **编号格言**（在《哲学研究》中为编号段落）
- **想象性对话**（"但这不是：……——不，不是这样。"）
- **反问和思想实验**
- **日常语言场景的反复观察**

#### 为什么要"示范"而非"论证"？
1. 如果哲学问题是混淆而非虚假信念，那么论证（提供理由）无法消除混淆，因为混淆者并不缺乏理由。
2. **"我给出的是提醒"（What I give is reminders）**（PI §127）——哲学工作不是揭示新的事实，而是提醒我们一直已知但被遗忘的语言使用方式。
3. **说服而非证明**：哲学的改变更像是"说服一个人采用不同的视角"，而非证明某个命题为真。

#### 与苏格拉底方法的区别
- 苏格拉底通过辩证推理迫使人承认矛盾。
- 维特根斯坦通过展示语言的多种用法，让人**自己看到**困惑如何产生。
- 最终目标：读者获得了处理类似困惑的能力，而不仅仅是接受了某个答案。

### 五、其他标志性方法论特征

| 特征 | 说明 |
|------|------|
| **反理论** | 哲学不产生理论、不解释、不假设——只描述 |
| **反本质主义** | 概念没有固定本质，只有家族相似 |
| **语境主义** | 意义总是依赖于使用的语境 |
| **反科学主义** | 哲学不是科学的延续或基础——它们是不同的活动 |
| **语言作为现象** | 关注语言实际运作的方式，而非其应然的逻辑结构 |
| **多元方法** | "没有单一方法，但有多种方法" |
