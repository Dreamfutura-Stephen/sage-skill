# Changelog

## [1.0.0] - 2026-05-02

### Added
- 完整的 Phase 0-4 执行流程：入口分流 → 目录创建 → 六维并行采集 → 框架提炼 → 档案构建 → 质量验证
- Phase 0B 诊断路径：用户不知该找哪位智者时，从 8 个困境维度反推推荐
- 六维 Agent 并行采集系统，每个 Agent 有明确搜索方向和输出规格
- 四重验证机制：文本一致性、逻辑自洽性、时代原创力、跨时空生成力
- 三重筛选机制：心智模型、决策启发式、丢弃的分级标准
- 双重检查点机制（调研质量 + 提炼确认）
- Phase 4 质量验证：已知测试、边缘测试、风格测试，含 7 项通过标准
- 信息源策略：5 层优先级、白名单、黑名单
- 特殊场景处理：历史智者 vs 近现代智者、东方智者 vs 西方智者
- 禁区规则：不讨论政治敏感话题、不编造原典、不包装通用道理、不忽略负面信息、不强行生成
- 苏格拉底对话协议四层递进 + 6 条黄金法则
- 14 位首批推荐智者名单
