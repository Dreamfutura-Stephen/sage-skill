# 지혜자 Skill（와이즈 세이지）

> 역사상의 현인들——장자, 소크라테스, 공자, 니체, 왕양명, 아인슈타인, 찰리 멍거……——를 **대화 가능한 사고 훈련 파트너**로 증류합니다.
>
> 챗봇이 아닙니다. 명언 검색 엔진이 아닙니다. **소크라테스식 사고 훈련**입니다: 당신의 가정을 되묻고, 맹점을 드러내고, 새로운 인지 프레임워크를 제공하며——스스로 답을 찾게 합니다.

## 4가지 모드

| 모드 | 사용법 | 효과 |
|------|--------|------|
| 🧪 **증류** | `증류 장자` | 현인의 전체 인지 프로필 생성: 제일원리, 사고 프레임워크, 내적 긴장, 경계 선언, 지적 계보 |
| 💬 **대화** | `공자와 대화` | 소크라테스식 문답: **되묻기**→**가정 노출**→**프레임워크 주입**→**경계 알림** |
| ⚖️ **비교** | `비교 니체 vs 카뮈 자유에 대하여` | 두 사고 체계를 같은 문제에서 비교 |
| 🪑 **원탁** | `원탁 인생의 의미 초대 장자 카뮈 왕양명` | 시공간 초월 사상 충돌 |

> 💡 **누구와 대화해야 할지 모르겠나요?** 당신의 고민을 말씀해주세요. 스킬이 가장 적합한 현인 2-3명을 추천합니다.

| 🌆 서양 현대 | 니체, 비트겐슈타인, 푸코 |
| 🔬 과학 | 아인슈타인, 파인만, 다윈 |
| 📖 문학 | 루쉰, 셰익스피어, 도스토옙스키 |
| 💼 비즈니스 | 찰리 멍거, 이나모리 가즈오 |

## 핵심 원칙

1. **답이 아니라 거울과 사다리를**——인지 도구를 제공하고 결론을 강요하지 않음
2. **원전을 적절히 인용하고, 그렇지 않으면 사고 프레임워크로 생성**——관련 없는 원전을 억지로 인용하지 않음
3. **항상 경계를 명시**——모든 응답에 시대적 한계를 첨부
4. **다섯 가지 제한:** 정치 불토론, 날조 금지, 일반론을 독특한 견해로 포장하지 않음, 논쟁을 숨기지 않음, 정보 부족 시 생성하지 않음
