## 4가지 모드

| 모드 | 사용법 | 효과 |
|------|--------|------|
| 🧪 **증류** | `증류 장자` | 현인의 전체 인지 프로필 생성 |
| 💬 **대화** | `공자와 대화` | 소크라테스식 문답 |
| ⚖️ **비교** | `비교 니체 vs 카뮈 자유에 대하여` | 두 사고 체계 비교 |
| 🪑 **원탁** | `원탁 인생의 의미 초대 장자 카뮈 왕양명` | 시공간 초월 사상 충돌 |

npx skills add Dreamfutura-Stephen/Sage-skill
