## Four Modes

| Mode | Usage | Effect |
|------|-------|--------|
| 🧪 **Distill** | `distill Socrates` | Generate complete cognitive profile (first principles, thinking frameworks, tensions, boundaries) |
| 💬 **Dialogue** | `talk to Confucius` | Socratic Q&A: rebound → expose assumptions → inject framework → boundary reminder |
| ⚖️ **Compare** | `compare Nietzsche vs Camus on freedom` | Two thinkers side by side on the same question |
| 🪑 **Roundtable** | `roundtable "meaning of life" invite Zhuangzi Camus Wang Yangming` | Cross-temporal intellectual collision |
