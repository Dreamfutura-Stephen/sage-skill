# 智者 Skill（Wise Sage）

> 智者 Skill 將歷史上的智者——莊子、蘇格拉底、孔子、尼采、王陽明……——蒸餾為**可對話的思維訓練夥伴**。
>
> 不是聊天機器人。不是語錄搜尋引擎。而是**蘇格拉底式的思維陪練**：反問你的假設、暴露你的盲區、給你新的認知框架——然後讓你自己找到答案。

## 四種模式

| 模式 | 用法 | 說明 |
|------|------|------|
| 🧪 **蒸餾** | `蒸餾 莊子` | 生成完整的智者認知檔案：第一性原理、核心思維框架、內在張力、邊界聲明、智識譜系 |
| 💬 **對話** | `與 孔子 對話` | 蘇格拉底式問答，四層遞進：**反彈**（讓你先說）→ **暴露假設**（挑戰未檢驗的前提）→ **框架注入**（用智者思維模型重新審視）→ **邊界提醒**（永遠揭露時代局限） |
| ⚖️ **比較** | `比較 尼采 vs 卡繆 關於自由` | 兩種思維框架放在同一問題下比較——哪裡一致、哪裡分歧 |
| 🪑 **圓桌** | `圓桌 人生的意義 邀請 莊子 卡繆 王陽明` | 跨時空思想碰撞——多位智者從各自框架回應同一問題 |

> 💡 **不知道找誰聊？** 直接說出你的困惑，Skill 會推薦 2–3 位最適合的智者。

## 覆蓋智者（18 位）

| 領域 | 智者 |
|------|------|
| 🏯 東方哲學 | 老子、莊子、孔子、王陽明 |
| 🏛️ 西方古典 | 蘇格拉底、柏拉圖、亞里斯多德 |
| 🌆 西方現代 | 尼采、維根斯坦、傅柯 |
| 🔬 科學思想 | 愛因斯坦、費曼、達爾文 |
| 📖 文學洞見 | 魯迅、莎士比亞、杜斯妥也夫斯基 |
| 💼 商業智慧 | 查理·芒格、稻盛和夫 |

## 核心理念

1. **不給答案，給鏡子和梯子**——你的困惑只有你自己能解開
2. **合適時引用原典，不合適時按思維框架生成**——不強行塞入不相關原典
3. **永遠暴露邊界**——每次回答都附帶時代局限
4. **五條硬限制：** 不碰政治、不編造、不包裝通用道理、不隱藏爭議、資訊不足不強行生成
