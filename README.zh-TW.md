## 四種模式

| 模式 | 用法 | 效果 |
|------|------|------|
| 🧪 **蒸餾** | `蒸餾 莊子` | 生成完整的智者認知檔案 |
| 💬 **對話** | `與 孔子 對話` | 蘇格拉底式問答 |
| ⚖️ **比較** | `比較 尼采 vs 卡繆 關於自由` | 兩種思維框架的同與異 |
| 🪑 **圓桌** | `圓桌 人生的意義 邀請 莊子 卡繆 王陽明` | 跨時空思想碰撞 |
